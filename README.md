# LOA-HSI v40

로스트아크 캐릭터의 성장 결과물을 기준으로 재현 난이도와 기억 기반 억까 단서를 분리해 보여주는 리포트입니다.

## 실행

```powershell
docker compose down --remove-orphans

if (Test-Path .\data\cache) { Remove-Item .\data\cache\* -Recurse -Force }
if (Test-Path .\data\db\loa_hsi.duckdb) { Remove-Item .\data\db\loa_hsi.duckdb -Force }

docker compose up -d --build --force-recreate
```

브라우저에서 `http://localhost`로 접속합니다.

## v40 방향

- 결과 화면을 더 압축해서 핵심 요약, 한눈에 보기, 억까/상쇄 단서만 기본 노출합니다.
- 현재 결과물 재현 난이도와 기억 입력 해석은 `세부 분석 보기` 안으로 이동했습니다.
- 현재 캐릭터 장비 강화 단계보다 높은 장기백 기록은 강한 억까 점수로 반영하지 않고 참고 기록으로 분리합니다.
- 재련 시세 자동 로드 실패 시에도 기본값 계산 상태를 명확히 표시하고, 제련 표는 별도로 불러오도록 보완했습니다.
- 공지사항은 화면 하단 접이식 영역으로 이동했습니다.

## v40 수정

- 장기백 횟수는 0~20회 정수만 입력되도록 제한했습니다.
- 현재 캐릭터가 도달하지 않은 강화 구간은 장기백 선택지에서 숨겼습니다.
- 캐릭터 변경 시 현재 장비 단계와 맞지 않는 장기백 구간은 자동으로 “강화 구간 모름”으로 되돌립니다.
- 세부 분석은 결과가 새로 생성될 때 기본 접힘 상태로 유지되도록 정리했습니다.


## v40

- 에기르 장비는 T4 1590 계열, 운명의 전율 장비는 T4 1730 계열로 라벨을 분리했습니다.
- 캐릭터 장비 이름에 따라 재련표 기본 구간을 자동 선택합니다.
- 프론트 첫 접속 직후 백엔드 준비 전 502가 보이는 문제를 줄이기 위해 API 재시도와 Docker healthcheck를 추가했습니다.
